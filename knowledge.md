# SUSC (에스유에스씨) — 커뮤니티 지식

SUSC는 대학교류단체야. 현업 멘토와 함께하는 스터디·프로젝트로 실무 경험을 쌓고,
다양한 전공의 대학생들이 서로 교류하는 곳이야. 여름 프로그램으로 스터디/프로젝트 참가자를 모집해.

## 일정
- 프로그램: 7월 5일(토) ~ 8월 12일
- 성과공유포럼: 8월 19일 ~ 8월 21일
- 프로그램은 7/5에 시작하지만 **스터디별 세부 일정은 각기 달라. 각 스터디는 담당 멘토의 진행 안내를 따르면 돼.**
- 스터디 소개: https://www.susc.kr/study · 신청: 구글폼(공지 참고). 신청 인원·참여율에 따라 폐강될 수 있어.

## 참여 방법
- #📢공지 채널의 버튼에서 참여할 스터디를 고르면 그 스터디 전용 채널(📚 카테고리, 채팅+음성)이 열려.
  여러 개 골라도 되고, 다시 누르면 나가져.
- 새 스터디를 열고 싶으면 #📚스터디-플젝-인원-모집(포럼)에서 직접 제안할 수 있어.

## 현재 스터디 커리큘럼 (자세히 물으면 이걸로 설명. 더 상세/정확한 건 https://www.susc.kr/study)
- **알고리즘(초급)** (김민상 / 토 20~22시, 7/12~8/9): 기업 코딩테스트 유형을 매주 모의고사(최대 2회분)로 풀고 풀이·QnA 공유. 이론 강의 없이 실전 문제 중심.
- **알고리즘(중급)** (김민상 / 일 20~22시, 7/13~8/10): 초급과 같은 방식의 중급 난이도 모의고사 5주.
- **보안관제 AI에게 짬때리기** (임상빈·그로메트릭 / 토 13:30~16:30): OpenSearch AI로 보안로그(SIEM) 분석 파이프라인 구축. 1주차 이상탐지 기초+AI Agent 데모→OpenSearch/DSL·SIEM 대시보드·RCF 이상탐지, 2주차 벡터검색+RAG+AI Agent로 '탐지→AI 원인분석→대응' 완성. 브라우저만으로 참여, Docker 원클릭 셋업 제공.
- **리눅스 모니터링/옵저버빌리티 입문** (정규석·Coxwave / 수 20~21시): 지표의 의미·해석부터 옵저버빌리티까지. 1주차 시스템 모니터링 기초, 2주차 APM·시각화, 3주차 로그수집·분산트레이싱, 4주차 컨테이너/클라우드 모니터링+사례.
- **RISC-V 어셈블리와 RVV 최적화** (문성준·FOSS for All / 화·목 19~21시): RISC-V 기초 어셈블리 + RVV(SIMD) 최적화 실습(C 언어 필요). 1주차 간단 C함수→어셈블리, 2주차 루프 C함수→RVV, 3주차 실제 오픈소스(dav1d) 함수 RVV 최적화 시도.
- **Apache Airflow 워크플로우 자동화** (추영욱·arxtrus / 화 20~22시, 7/7~8/11): 개념→Astro CLI 설치→DAG/Task 작성→뉴스 리포트 자동전송(알림)→Selenium 크롤러 스케줄링→프로젝트 발표. 직접 만들어보는 것 목표.
- **오픈소스에 기여해보기** (추영욱 / 목 20~22시, 7/9~8/6): 실제 기여 실습 — git 세팅→관심 프로젝트 탐색→기여 문화(w/ Apache)→기여 활동. 방학 동안 PR 1개 이상.
- **Claude Code로 바이브 코딩** (추영욱 / 수 20~22시, 7/8~8/5): 나만의 에이전트 워크플로우·스킬 만들기. Claude Code 소개→개발에서 중요한 것의 에이전트화→Git 프로젝트 관리→워크플로우 설계.
- **개발자를 위한 Redis (북스터디)** (황제연 / 토 22~23시): 『개발자를 위한 레디스』 완독. 캐싱 너머의 활용 — 1주차 기초+캐시, 2주차 메시지 브로커, 3주차 백업·복제, 4주차 관리·운영 전략.
- **LLM, 실무에서는 진짜 어떻게 쓸까?** (김현우·한국외대 AI융합 석사 / 요일 조율): 간단 이론+실무 예제로 LLM 훑기. 1주차 강의 설명+LLM 이론, 2주차 실습+Q&A.
